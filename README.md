# Cronómetro y Temporizador Online Gratis

Aplicación web gratuita con **cronómetro de precisión** y **temporizador con cuenta regresiva**.

## ✨ Características

- **Cronómetro** con precisión de centésimas.
- Registro de **vueltas (laps)**.
- **Temporizador** con cuenta regresiva personalizable.
- Funciones: Iniciar, Pausar, Continuar y Reiniciar.
- Diseño oscuro elegante.

## 🚀 Uso

1. Elige modo: **Cronómetro** o **Temporizador**.
2. Pulsa **Iniciar** para comenzar.
